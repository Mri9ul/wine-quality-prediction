# Wine Quality Prediction using Random Forest

## Overview
This project predicts the quality of wine based on its chemical properties.

## Problem Type
Regression problem (predict numerical quality score)

## Features Used
fixed acidity
volatile acidity
citric acid
residual sugar
chlorides
free sulfur dioxide
total sulfur dioxide
density
pH
sulphates
alcohol

## Model
Random Forest Regressor

## Evaluation Metrics
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
R2 Score

## Results
The model captures general trends in wine quality, though exact prediction is difficult due to subjective nature of taste.

## Tools Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
